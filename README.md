## Getting Started

1. Clone the repository
2. Open `index.html` in your web browser
3. The authentication gate will appear first - login or create an account to access the site

## Features in Detail

### Authentication
- Login/Create Account modal
- Email and password validation
- Persistent authentication state
- Required before accessing the website

### Shopping Cart
- Add items to cart
- View cart in sidebar
- Remove items
- Calculate total
- Checkout functionality

### Wishlist
- Add items to wishlist
- View wishlist in sidebar
- Remove items
- Add items directly to cart from wishlist

### Theme System
- Dark and light themes
- Smooth transitions
- Persistent theme preference
- Theme toggle button

### Form Validation
- Real-time validation
- Inline error messages
- Required field validation
- Email format validation
- Phone number validation
- Order number validation
