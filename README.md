# QZ Stores - E-commerce Website

A modern, responsive e-commerce website built with HTML, CSS, and JavaScript.

## Features

-  **Authentication System** - Login/Create Account gate with form validation
-  **Shopping Cart** - Add/remove items with persistent storage
-  **Wishlist** - Save favorite products for later
-  **Dark/Light Theme** - Toggle between themes with persistent preference
-  **Responsive Design** - Works on all devices
-  **Form Validation** - Contact form with comprehensive validation
-  **Modern UI** - Clean, minimalist design with smooth animations

## Pages

- **Home** (`index.html`) - Product listings with category filters
- **About** (`about.html`) - Company information and team
- **Help** (`help.html`) - FAQ and support information
- **Contact** (`contact.html`) - Contact form and information

## Technologies Used

- HTML5
- CSS3 (with CSS Variables for theming)
- JavaScript (ES6+)
- LocalStorage for data persistence

## Getting Started

1. Clone the repository
2. Open `index.html` in your web browser
3. The authentication gate will appear first - login or create an account to access the site

## Features in Detail

### Authentication
- Login/Create Account modal
- Email and password validation
- Persistent authentication state
- Required before accessing the website

### Shopping Cart
- Add items to cart
- View cart in sidebar
- Remove items
- Calculate total
- Checkout functionality

### Wishlist
- Add items to wishlist
- View wishlist in sidebar
- Remove items
- Add items directly to cart from wishlist

### Theme System
- Dark and light themes
- Smooth transitions
- Persistent theme preference
- Theme toggle button

### Form Validation
- Real-time validation
- Inline error messages
- Required field validation
- Email format validation
- Phone number validation
- Order number validation
